# 📚 学习笔记管理系统

这是我创建的第一个前端项目！一个可以记录学习笔记的网页应用。

## 🌟 功能特点
- ✅ 添加、查看、编辑笔记
- ✅ 笔记搜索功能  
- ✅ 标签分类系统
- ✅ 本地数据保存
- ✅ 响应式设计（支持手机）

## 🚀 在线体验
GitHub Pages: [点击这里体验](https://你的用户名.github.io/learn-notes)

## 🛠️ 技术栈
- HTML5
- CSS3  
- JavaScript
- LocalStorage
