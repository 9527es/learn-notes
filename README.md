# 📚 学习笔记管理系统

这是我创建的第一个前端项目！一个可以记录学习笔记的网页应用。

## 🌟 功能特点
- ✅ 添加、查看、编辑笔记
- ✅ 笔记搜索功能  
- ✅ 标签分类系统
- ✅ 本地数据保存
- ✅ 响应式设计（支持手机）

## 🚀 在线体验
GitHub Pages: [点击这里体验](https://你的用户名.github.io/learn-notes)

## 🛠️ 技术栈
- HTML5
- CSS3  
- JavaScript
- LocalStorage

## 📁 项目结构
# 进入项目文件夹（根据你的实际位置调整）
cd Desktop\learn-notes

# 初始化Git
git init

# 添加所有文件
git add .

# 提交到本地
git commit -m "第一次提交：创建学习笔记项目"

# 连接到GitHub（复制你仓库的地址）
git remote add origin https://github.com/你的用户名/learn-notes.git

# 上传到GitHub
git push -u origin main
